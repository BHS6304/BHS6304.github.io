We have to use iframe code which works fine on laptops and computers but hides the recording list behind the player window on phones and small tablets. A website pro could fix the problem. To verify, use your phone to go to our live stream page. Then use your phone on the page at the link below and see the difference. Another benefit of using JS is that we can list more than 5 previous recordings per page. We can set it to however many we choose.
